Creating Tables

To create a table use the table tag. In table tag there are 3 portions -

  1. Table Head
  2. Table Body
  3. Table Foot

Table head will be used to define the heading row. To use it, thead in angular brackets is written. To define the heading data, use tr tag ( table row ) and use th tag ( table heading ) and finally you can enter data.

Table body will be used to write the data of the table. To use it, tbody is used. Then Use tr tag for the row and then use td tag ( table data ) to write the information.

Table foot is used to define summary of the table. To use it, tfoot will be used followed by tr and td respectively.

Here is a sample table code -

Name Age Salary
Rohit 27 10000
Mohit 25 20000
Riya 23 100000
