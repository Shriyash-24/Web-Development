Lists

There are two types of Lists -

  • Unordered Lists
  • Ordered Lists

Unordered Lists

In unordered lists, order does not matter. By default a bulleted list is created. To change bulleted lists to something else, attributes are used. To use it, (angular bracket)ul style="list-style-type: square/disc/circle are used. You can make it square,disc,circle. This image will describe it better -

Ordered Lists

Ordered Lists are used when we would like to follow things in order. Ordered lists hold a semantic meaning. By default numbered list is created. To change numbered lists into something else, ol type="a" is used. You can use lowercase, uppercase alphabets, numbered list, lowercase and uppercase roman numerals.
