Image Tag

It is a self closing tag used for posting images in the webpage. The src or source attribute contains the source of the image, we can also use it to set height and width of the image and an alternate text is provided at the end for screen readers and SEO. If image does not load, the alternate text loads. There can be many reasons why image is not loading -

  • Because of Website
  • User authentication requirement
  • Bandwidth reasons.
